一、AI 產業重點新聞

**1.**

- 主題：OpenAI 發布 GPT-5.6 系列模型 Sol、Terra、Luna
- 功能：GPT-5.6 為 OpenAI 最新旗艦模型家族，於7月9日正式公開發布。Sol 主攻前沿推理與長時間代理任務，Terra 為平衡型日常模型，Luna 為最低成本高速模型。Sol 在 Agents' Last Exam 評測中取得53.6分，超越 Claude Fable 5 達13.1分。
- 成長幅度：定價策略大幅降低，Sol 為每百萬 token 輸入5美元/輸出30美元，Terra 輸入2.5美元/輸出15美元，Luna 輸入1美元/輸出6美元。已全面開放於 ChatGPT、Codex 與 OpenAI API。
- 可應用領域發想：長時間自動化工作流程、多步驟程式開發代理、企業知識工作自動化。Terra 的性價比適合中小型團隊快速導入。

- 功能：GLM-5.2 於6月17日以 MIT 授權開源發布，支持100萬 token 上下文窗口，在長時間程式開發任務中僅落後 Claude Opus 4.8 約1%，同時超越 GPT-5.5。原生支持華為昇騰等國產晶片。
- 成長幅度：定價僅為美國前沿模型的十分之一，輸入1.4美元/輸出4.4美元每百萬 token。引發西方國家對中國 AI 競爭力的重新評估。
- 可應用領域發想：企業可用於私有化部署的高性能 LLM 方案，特別適合對資料安全要求高且預算有限的團隊。開源授權允許完全商業使用。

- 主題：AI 產業從追求模型規模轉向實用性與可靠性
- 功能：2026年7月標誌著AI產業焦點從「模型有多大」轉向「模型能否在無監督下完成真實任務」。業界開始重視成本效率、任務完成率與長時間穩定性，而非單純的基準測試分數。

- 可應用領域發想：開發團隊應重新評估模型選擇策略，針對不同任務場景選用不同等級模型，而非一律使用最昂貴的旗艦模型。

二、GitHub 熱門專案

**1.**

- 主題：MadsLorentzen/ai-job-search
- 功能：基於 Claude Code 的 AI 求職框架，能自動評估職缺、客製化履歷、撰寫求職信並準備面試。採用 drafter-reviewer 代理架構，將求職申請視為對抗性文件處理，自動驗證關鍵字與 ATS 解析結果的一致性。
