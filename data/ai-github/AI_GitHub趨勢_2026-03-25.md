一、AI 產業最新動態 1. OpenAI 營收突破 250 億美元，考慮上市 

OpenAI 年化營收已超過 250 億美元，據報導正在為 2026 年底的 IPO 做初步準備。競爭對手 Anthropic 也接近 190 
億美元年化營收。
2. NVIDIA 推出 Agent Toolkit 開源平台 

NVIDIA 發布 Agent Toolkit，包含 OpenShell 安全執行環境與 Nemotron 模型，專為企業開發自主 AI 
代理。Adobe、Atlassian、Salesforce 等大廠已開始採用。
3. OpenAI 開源青少年安全工具 

OpenAI 於 3/24 發布開源青少年安全提示集與 gpt-oss-safeguard 安全模型，協助開發者內建青少年保護機制。
4. 開源模型競爭白熱化：GLM-5 與 Qwen 3.5 

智譜 AI 的 GLM-5 以 MIT 授權推出達前沿水準；阿里巴巴 Qwen 3.5 Small 以 Apache 2.0 發布四款多模態模型。
5. 迪士尼全面導入生成式 AI 

迪士尼正式將生成式 AI 嵌入整體營運架構，涵蓋內容創作、後製流程與個人化遊客體驗。
