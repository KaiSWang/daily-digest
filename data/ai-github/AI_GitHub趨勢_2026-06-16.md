**1.**
- 主題：Model Context Protocol（MCP）
- 功能：由 Anthropic 主導、Linux Foundation 治理的 AI 連接標準，提供統一的 AI 主機與外部工具整合介面，所有主要 AI 廠商（OpenAI、Google DeepMind、Microsoft）已相容，支援資料庫、檔案系統、訊息系統等超過 500 個公開伺服器，被稱為「AI 開發的 USB-C 標準」。
- 成長幅度：2026 年 3 月突破 9,700 萬次安裝，規格持續進化，7 月版本候選引入無狀態核心與 OAuth 對齊的授權機制
- 可應用領域發想：台灣 SaaS 企業可透過建立 MCP Server 讓企業產品直接整合主流 AI 助理，作為差異化競爭優勢；開發者可將既有 API 快速包裝為 MCP 服務，進入 AI 生態系市場。
