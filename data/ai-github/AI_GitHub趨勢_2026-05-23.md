二、GitHub 熱門專案（至少 5 個）

**1.**
- 主題：OpenClaw
- 功能：開源個人 AI 助理，可在自有硬體上運行，整合 WhatsApp、Telegram、Slack、Discord、Signal、iMessage、Microsoft Teams 等主流通訊管道，ClawHub 技能市集已累積超過 5,700 個社群技能，涵蓋行事曆管理到程式碼審查。
- 成長幅度：總星數突破 300,000，1 月病毒式傳播後從 9,000 飆升，持續創下單週最高成長紀錄。
- 可應用領域發想：企業內部知識庫 Bot、個人化 AI 秘書、多平台客服自動化整合。
- 連結：https://trendshift.io/

**2.**
- 主題：Browser Use
- 功能：讓 AI Agent 能夠操作瀏覽器，成為 2026 年開源瀏覽器自動化事實標準，支援主流 LLM 接入，可執行網頁爬取、表單填寫、動態內容擷取等任務。
- 成長幅度：GitHub 星數達 92,000，持續為 AI Agent 基礎設施最核心依賴之一。
- 可應用領域發想：自動化資料蒐集、電商價格監控、政府資訊填報自動化、RPA 替代方案。
- 連結：https://github.com/browser-use/browser-use

**3.**
- 主題：Ollama
- 功能：以極簡 CLI 介面在本地端運行各類大型語言模型，支援 Llama、Mistral、Gemma 等主流模型，無需雲端 API 即可完成推論。
- 成長幅度：GitHub 星數突破 162,000，為本地 LLM 部署最廣泛使用工具。
- 可應用領域發想：企業私有化部署、醫療與法律資料不得外傳場景、離線環境邊緣推論、開發測試環境成本優化。
- 連結：https://github.com/ollama/ollama

**4.**
- 主題：RAGFlow
- 功能：專為企業知識庫設計的 RAG（Retrieval-Augmented Generation）基礎設施，支援多文件格式與多資料來源，提供合規友善的知識管理與 AI 問答能力。
- 成長幅度：GitHub 星數超過 70,000，為企業 RAG 架構最熱門選擇之一。
- 可應用領域發想：企業內部法規遵循查詢、研究機構文獻助理、多源異質資料知識融合平台。
- 連結：https://github.com/infiniflow/ragflow

**5.**
- 主題：Mastra
- 功能：以 TypeScript 為核心的 Opinionated AI Agent 框架，內建 RAG、可觀測性（Observability）、MCP 整合與視覺化工作流程建構器，降低 Agent 系統開發門檻。
- 成長幅度：GitHub 星數超過 21,000，為 TypeScript 生態系中成長最快的 Agent 框架。
- 可應用領域發想：前端工程師快速構建 AI 產品、企業後台自動化流程、結合 MCP 工具集的多步驟 Agent 應用開發。
- 連結：https://github.com/mastra-ai/mastra

**6.**
- 主題：GenericAgent
- 功能：自我進化 Agent 框架，從 3,300 行種子程式碼出發，透過技能樹自主成長，實現完整系統控制，token 消耗量較同類框架減少 6 倍，引入基因組進化協議（GEP）新範式。
- 成長幅度：單日新增星數打破歷史紀錄達 1,131 顆，為近期最受關注的自進化 Agent 框架。
- 可應用領域發想：長期任務自主執行、系統管理自動化、持續學習型個人助理、研究型 Agent 應用。
- 連結：https://github.com/lsdefine/GenericAgent

三、開發者工具與框架趨勢（N 項）

**1.**
- 主題：Claude Code
- 功能：Anthropic 推出的終端原生 AI 程式設計代理，深度整合 MCP（Model Context Protocol），SWE-bench 得分 78.4% 為主流工具最高，支援多 Agent 並行工作流程與背景執行模式。
- 成長幅度：2026 年 H1 已成為開發者使用量最高的 AI 程式設計工具之一，與 Cursor 合計佔 AI 程式設計工具市場超過 60% 的開發者使用份額。
- 可應用領域發想：大型程式碼庫重構、複雜 Bug 定位與修復、多檔案跨專案開發自動化、CI/CD 流程 AI 強化。
- 連結：https://www.digitalapplied.com/blog/ai-coding-agents-claude-code-cursor-codex-replit-2026

**2.**
- 主題：MCP（Model Context Protocol）
- 功能：由 Anthropic 主導的開放連接器標準，2026 年 5 月已成為 AI 工具與資料來源之間的事實標準，各主流 IDE 與 Agent 框架皆已內建 MCP 支援，取代過去各自為政的 API 整合方式。
- 成長幅度：相容 MCP 的工具生態在 2026 Q2 已超過 2,000 個，季增超過 300%。
- 可應用領域發想：企業內部工具統一接入 LLM、多 Agent 系統間標準化溝通、開發者工具鏈快速 AI 化。
- 連結：https://thenewstack.io/ai-coding-tool-stack/

**3.**
- 主題：Cursor 3
- 功能：IDE 整合型 AI 程式設計工具，以最快的行內補全速度著稱，2026 年新版本加入 Agent 模式與背景工作流程，與 Claude Code 形成互補使用的主流雙工具組合。
- 成長幅度：開發者平均同時使用 2.3 款 AI 程式設計工具，Cursor 與 Claude Code 為最常見配對。
- 可應用領域發想：日常程式撰寫效率提升、即時程式碼審查、快速原型開發、大規模程式碼搜尋與替換。
- 連結：https://www.digitalapplied.com/blog/ai-coding-h1-2026-retrospective-cursor-claude-code-codex-data
