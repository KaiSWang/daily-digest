二、GitHub 熱門專案

**1.**

- 主題：OpenCut — 開源影片剪輯器
- 功能：MIT 授權的跨平台影片編輯器，支援 Web、桌面與行動裝置。以 Next.js 為前端、Rust 核心處理 GPU 合成與特效，2026 年 5 月完成全面重寫，採用插件優先架構，計畫支援 MCP 與 headless API。
- 成長幅度：GitHub 星數達 75,000 顆，近期穩居 GitHub Trending 第一名，一年內從零成長至當前規模
- 可應用領域發想：取代 CapCut 等付費工具的自架方案、企業內部影片製作流水線、AI 影片後製自動化整合
- 連結：https://www.google.com/url?q=https://github.com/OpenCut-app&source=gmail&ust=1784932630066000&sa=E

**2.**

- 主題：OpenCode — 開源終端 AI 程式開發代理
- 功能：MIT 授權的終端程式開發代理，由 SST 團隊開發，可連接 75 種以上 AI 模型供應商，完全在本地執行，開發者可自由選擇模型、自行部署。
- 成長幅度：GitHub 星數突破 160,000 顆，月活躍開發者約 750 萬人，在 SpaceX 收購 Cursor 後成為開發者首選替代方案
- 可應用領域發想：不依賴特定廠商的 AI 開發環境、企業內部程式碼審查自動化、CI/CD 流程中的 AI 輔助

**3.**

- 主題：Orca — 平行 AI 程式代理協調環境
- 功能：開源 Agent Development Environment，可同時協調多個 AI 程式代理（Claude Code、Codex CLI、OpenCode 等 20 種以上），使用開發者自有的 API 訂閱，無按席計費、無廠商鎖定，支援無限平行運算。
- 成長幅度：GitHub 星數超過 12,500 顆，獲得 Y Combinator 投資，2026 年 7 月持續位居 Trending 前列
- 可應用領域發想：大型專案多代理協同開發、降低 AI 開發工具訂閱成本、跨模型比較測試
- 連結：https://www.google.com/url?q=https://github.com/VirtusLab/orca&source=gmail&ust=1784932630066000&sa=E

**4.**

- 主題：CubeSandbox — AI 代理輕量級安全沙箱
- 功能：騰訊雲開源的 AI 代理沙箱，提供毫秒級啟動（平均冷啟動低於 60ms）、硬體級隔離、每實例不到 5MB 記憶體開銷，專為 AI 代理安全執行設計。
- 成長幅度：發布以來迅速進入 GitHub Trending，填補了 AI 代理基礎設施中「安全執行環境」的空白
- 可應用領域發想：企業 AI 代理生產環境部署、多租戶 AI 服務隔離、程式碼執行沙箱即服務

- 功能：專為 AI 程式代理設計的終端多工器，提供持久化工作空間、分頁、分割視窗，以及 CLI 程式代理的狀態偵測功能。
