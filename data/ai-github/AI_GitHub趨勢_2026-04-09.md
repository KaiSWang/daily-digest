- 成長幅度：從 9,000 星爆衝至 210,000+ 星，被形容為 GitHub 歷史上成長最快的開源專案之一。
- 可應用領域發想：台灣中小企業可在內部電腦部署 OpenClaw，建構不外洩資料的私人 AI 助理，整合內部 IM 與工作流程。
